# 📰 AI 기반 맞춤형 뉴스 요약 서비스

이 프로젝트는 **Next.js**와 **AWS(Amplify, Bedrock, Lambda)**를 활용한 AI 뉴스 요약 서비스입니다.
팀원분들은 아래 절차에 따라 개발 환경 설정을 진행해 주세요.

---

## 🛠 팀원 개발 환경 설정 가이드

### 1. 필수 소프트웨어 설치
* **Node.js**: `v24.12.0` 버전 권장 (버전이 너무 낮으면 오류가 발생할 수 있습니다.)
* **Git**: 설치 및 GitHub 계정 연동 필요

### 2. 프로젝트 내려받기 및 라이브러리 설치
터미널(CMD)을 열고 아래 명령어를 순서대로 입력하세요.

```bash

git clone [https://github.com/LibrumLego/ai-news-summarizer.git](https://github.com/LibrumLego/ai-news-summarizer.git)

# 2. 프로젝트 폴더로 이동
cd ai-news-summarizer

# 3. 노드 패키지(라이브러리) 설치
